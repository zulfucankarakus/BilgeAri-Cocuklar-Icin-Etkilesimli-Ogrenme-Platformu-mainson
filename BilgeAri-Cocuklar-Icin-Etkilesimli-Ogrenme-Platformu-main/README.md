# BilgeAri Cocuklar icin Etkilesimli Ogrenme Platformu
Çocukların dijital dünyada güvenle, eğlenerek öğrenmesini amaçladığımız bu web sitesinin içeriğinde, çocuklara yönelik ve onlar için sadeleştirilmiş anlatımlarıyla felsefe, kodlama, yabancı dil, matematik, doğa, tarih, bilim, oyun gibi konu başlıklarını içeren alanlar mevcuttur.
